Каталог одноименной серии игр Yakuza

1. Главная страница - сам каталог с карточками игр, которые можно добавить в корзину, кликнув на кнопку "купить"

2. Для добавления новых карточек на главную страницу в поисковой строке наберите /create. Откроется форма, которая автоматически отправляет полученные сведения в базу данных (Games). 
В форме необходимо указать название игры, ее стоимость (числа с запятой учитываются), описание и НАЗВАНИЕ ОБЛОЖКИ С УЧЕТОМ РАСШИРЕНИЯ (например, фон.png). Т.к. в БД нельзя загрузить картинку, в нее вносится название используемой и заранее подгруженной в папку static/my_app/images фотографии, чтобы в дальнейшем не возникло проблем с её подгрузкой на страницу корзины.

3. Добавленные в корзину товары автоматически заносятся в отдельную базу данных (Cart)

4. На странице корзины выводятся ранее выбранные для покупки игры. Причем, если одна и та же карточка была выбрана несколько раз, она не дублируется, а выводится лишь один раз с указанием количества ее выбора в верхнем правом углу карточки. 

5. Подсчет итоговой суммы приводится справа от карточек.
